# Tchalanet Edge Service

Service **Node/TypeScript** pour :

- évaluer des règles JSON (`json-rules-engine`),
- rendre des templates Liquid,
- envoyer des communications multi-canaux (WEB, SMS, EMAIL, WHATSAPP, PUSH),
- être appelé depuis `tchalanet-server` via des endpoints HTTP.

## Démarrage

```bash
npm install
npm run dev
```

Par défaut, le service écoute sur `PORT=4001`.
