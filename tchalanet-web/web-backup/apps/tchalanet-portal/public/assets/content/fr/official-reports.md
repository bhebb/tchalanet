> **Info :** Cette page centralise l’orientation vers les **récapitulatifs officiels** publiés par les autorités et opérateurs agréés.

## 1) Où consulter
- Portails officiels, communiqués et tableaux récapitulatifs (LEH / opérateurs).  
- Médias ou applications qui relaient les résultats — privilégier **la source officielle**.

> **Attention :** En cas de divergence de sources, **le dernier avis officiel** prévaut (corrections possibles).

## 2) Vérifier un ticket
Utilisez **/ticket/:code** pour consulter l’état public : **valide**, **en attente** (synchro) ou **annulé**.  
Vous pouvez aussi passer par **/verify** si vous ne connaissez pas l’URL exacte.

## 3) Comprendre les horaires
- Les horaires de tirage peuvent changer (jours fériés, calendrier local, décisions des autorités).  
- Fiez‑vous aux **annonces les plus récentes** publiées par les sources officielles.

## 4) Après le tirage : que se passe‑t‑il ?
- Les tickets gagnants suivent une **procédure de paiement** (montants, délais, vérifications).  
- Conservez votre ticket jusqu’à paiement effectif ; suivez les instructions de l’opérateur.
