# Politique de confidentialité

*Dernière mise à jour : juin 2026*

---

## 1. Introduction

Tchalanet s'engage à protéger la vie privée des utilisateurs de ses services publics. Cette politique décrit les informations que nous collectons, leur utilisation, et vos droits.

## 2. Informations collectées

### Informations de navigation

Lors de votre utilisation des surfaces publiques Tchalanet, nous collectons automatiquement des informations techniques telles que l'adresse IP, le type de navigateur, les pages visitées et la durée des sessions. Ces données sont utilisées uniquement à des fins d'analyse et d'amélioration du service.

### Code public de ticket

Lorsque vous utilisez la fonctionnalité de vérification de ticket, le code public que vous saisissez est transmis à notre service afin de consulter le statut du ticket correspondant. Ce code ne contient aucune donnée personnelle identifiable.

## 3. Utilisation des données

Les données collectées sont utilisées pour :

- Assurer le bon fonctionnement des services publics
- Analyser l'utilisation et améliorer l'expérience
- Détecter et prévenir les usages frauduleux
- Répondre aux obligations légales applicables

## 4. Partage des données

Tchalanet ne vend pas vos données personnelles. Les données peuvent être partagées avec des prestataires techniques opérant sous accord de confidentialité strict, uniquement dans le cadre de la fourniture des services.

## 5. Conservation

Les données de navigation sont conservées pour une durée maximale de 13 mois. Les requêtes de vérification de ticket ne sont pas associées à un profil utilisateur.

## 6. Vos droits

Conformément à la législation applicable, vous disposez des droits suivants :

- **Droit d'accès** : consulter les données vous concernant
- **Droit de rectification** : corriger des données inexactes
- **Droit à l'effacement** : demander la suppression de vos données
- **Droit d'opposition** : vous opposer à certains traitements

Pour exercer ces droits, contactez-nous via la page [Contact](/public/contact).

## 7. Cookies

Les surfaces publiques Tchalanet utilisent des cookies techniques nécessaires au fonctionnement du service. Aucun cookie publicitaire n'est utilisé.

## 8. Modifications

Cette politique peut être mise à jour. Toute modification significative sera notifiée via les canaux de communication officiels Tchalanet.

Pour toute question, contactez notre équipe via [la page Contact](/public/contact).
