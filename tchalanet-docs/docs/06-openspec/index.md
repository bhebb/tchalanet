# OpenSpec

OpenSpec is the planning and specification workspace. MkDocs exposes a map to
OpenSpec locations; it does not copy every proposal, task list, or component
spec into the published portal.

Rules:

- Global `openspec/context/` stays light and acts as a router.
- Component OpenSpecs stay in their owning component.
- Active changes are discoverable through links and generated inventory.
- Archived changes are discoverable but are not default AI context.
