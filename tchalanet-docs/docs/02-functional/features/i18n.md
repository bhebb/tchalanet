# Feature: i18n — Fonctionnel

## Rôle

Exposer les bundles i18n organisés par namespace, avec overrides tenant.

## Cross-apps (Web / Mobile)

### Web (UI)

- Pages: i18n charge au bootstrap
- i18n: `i18n.*`

### Mobile (POS)

- Écrans: chargement initial des bundles

### API (contrats)

- Endpoints:
  - `GET /api/v1/public/i18n/{locale}`
  - `GET /api/v1/app/i18n/{locale}`
