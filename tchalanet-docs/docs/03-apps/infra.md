# Infra

Infra operational detail remains in the infra project.

Canonical locations:

- `tchalanet-infra/docs/`
- `tchalanet-infra/CLAUDE.md`
- `tchalanet-infra/openspec/`
- `tchalanet-infra/envs/common/compose.env`

MkDocs should provide architecture and operations entrypoints, while infra
runbooks and environment details stay near the infra code.
