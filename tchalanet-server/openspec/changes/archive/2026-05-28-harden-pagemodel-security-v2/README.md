# harden-pagemodel-security-v2

This OpenSpec package fixes:

- public/private PageModel boundaries
- secure private PageModel resolution
- cashier as an operational private surface
- predictable shell/widget contracts
- `navigationDrawer` vocabulary
- typed image/link/action models
- regenerated shell fragments for public, cashier, tenant admin, and superadmin

Use this as the implementation guide for `features.pagemodel`.
