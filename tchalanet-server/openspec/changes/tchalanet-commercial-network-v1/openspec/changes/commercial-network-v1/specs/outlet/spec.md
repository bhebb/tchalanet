# Spec — Outlet

## ADDED Requirements

### Requirement: Outlet kind

The system SHALL classify outlets by kind: OWNED_SHOP, KIOSK, MOBILE_POINT, BANK_BRANCH, PARTNER_INSTITUTION, PARTNER_TENANT, REGIONAL_OFFICE.

### Requirement: Outlet remains operational channel

Outlet SHALL represent the channel/location/institution where sales occur. Outlet SHALL NOT represent seller identity.

### Requirement: Outlet sale eligibility

The system SHALL validate that an outlet is active and usable before a sale is created.
