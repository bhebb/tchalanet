# Tchalanet — Sécurité login, POS, mobile et transactions

Ce livrable contient la documentation fonctionnelle et technique pour sécuriser les connexions Web/Mobile/POS et les transactions Tchalanet, ainsi qu’un OpenSpec prêt à donner à un agent de développement.

## Objectif

Affirmer Tchalanet comme une plateforme viable de vente par téléphone et POS :

> Un login ouvre une session utilisateur.  
> Un terminal trusted ouvre une capacité de vendre.  
> Une session opérationnelle valide permet une transaction.
