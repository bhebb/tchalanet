# Spec delta: catalog.game

## ADDED Requirements

### Requirement: game_code is sold product only

`game_code` SHALL identify products sold to customers, such as `HT_BOLET`, `HT_MARYAJ`, or `HT_LOTO3/4/5`.

## REMOVED Requirements

### Requirement: US provider codes as game_code

US provider/internal codes SHALL NOT be used as `game_code`.
